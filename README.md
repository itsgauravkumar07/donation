# Donation Tracking System

A full-stack donation management platform built with React, Node.js, Express, and MongoDB.

## Features

### User

* View fundraising campaigns
* Donate to campaigns
* View donation dashboard
* View top donors leaderboard

### Admin

* Create campaigns
* Manage campaigns
* View donation statistics
* Monitor donations and fundraising progress

## Tech Stack

**Frontend**

* React.js
* React Router
* Tailwind CSS
* Axios

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB Atlas
* Mongoose

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```
